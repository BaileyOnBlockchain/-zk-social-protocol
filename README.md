A suite of Solidity smart contracts for a privacy-first, decentralised social protocol built on ZK proof primitives. Users can post, join groups, tip, stake, and govern anonymously without sacrificing verifiability.
The protocol centres on anonymous posting via Semaphore ZK proofs — identity is hidden, but authenticity is provable on-chain. Posts are content-hashed and linked to nullifier hashes, meaning each identity can only post once per context, preventing spam without revealing who anyone is. An optional EZKL integration allows ZK machine learning proof verification for on-chain toxicity moderation — the poster submits a proof that their content scores below a threshold, verified entirely on-chain without exposing the content itself.
Beyond the core posting layer, the suite covers the full protocol stack: ZK group membership via Semaphore, token-gated channel access for ERC20 and ERC721, an on-chain scam reporting and trust scoring system, soulbound-style profile NFTs with IPFS metadata, a Yearn-style staking vault, and an M-of-N multi-signature treasury with timelock enforcement.
13 contracts in total. Every state-changing function touching tokens or ETH is reentrancy-guarded. Treasury proposals enforce timelock delay even for single-signer configurations — no instant execution attacks. Gas optimisations applied throughout: struct packing, immutable variables, unchecked increments, calldata over memory where possible. A ZKsync Era optimised variant is included with packed storage slots.
This is a snapshot of a larger project. The contracts represent the on-chain layer only — frontend, ZK circuit setup, EZKL model proving, and deployment infrastructure are separate. There is a lot more where this came from.
